Mano Machine Assembler

An in-browser simulation of a theoretical 16-bit microcontroller as described in Computer System Architecture (3rd ed.) by Morris Mano. Interactive documentation is designed to learn assembly language in a hands-on setting. Built using TypeScript, React, Tailwind CSS.

Capitol Compass

A native iOS 17 app to view and explore Ottawa's landmarks and amenities through the MapKit interface. Uses Ottawa's public API to lazily pull data in real time using JSON queries to display in app. Built using SwiftUI.

Rate My Professor Extension

Chromium web extension to inject professors rating and description from  the RateMyProf GraphQL API to the uOttawa course building website. Built using JavaScript.

Cycling Club Android App

A CRUD Create Read Update Delete mobile app developed in a collaborative team environment. Utilized a Firebase Realtime Database for robust reading and writing capabilities, ensuring efficient data handling for user authentication and role-based access. Built using Java in Android Studio.

Grade Calcuator App

Tiny Raytracer

A compact, CPU-based ray tracing application designed for rendering (relatively) realistic 3D scenes. Features basic geometric shapes and implements essential ray tracing techniques for soft shadows, reflections, and refractions. Built in C.
